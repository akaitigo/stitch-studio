# PRD: stitch-studio

## 概要

クロスステッチの図案をブラウザ上のグリッドエディタで設計し、必要な糸の色番号・数量を自動算出するWebアプリケーション。画像をアップロードしてドット絵風の図案を自動生成する機能を持ち、Rust/WASMによる高速な画像処理をブラウザ内で実現する。

## 背景

- クロスステッチパターンツール市場は WinStitch / KG-Chart 等の古いデスクトップアプリが主流
- Web + WASM での現代的UIは明確な差別化要因
- 糸メーカー別のカラーパレット対応 + 買い物リスト自動生成は手芸愛好家の実用ニーズ

## ターゲットユーザー

クロスステッチ愛好家（初心者〜中級者）。PCにソフトをインストールせず、ブラウザで手軽に図案を作りたい層。

## MVP機能

### 1. 画像→グリッド変換エンジン (Rust/WASM)
- 画像ファイル（PNG/JPEG）を読み込み、指定サイズのグリッドに変換
- 色数削減アルゴリズム（中央値カット法 / k-means）で糸色数を制限
- ピクセル→グリッドセルへのマッピング

### 2. グリッドエディタ UI (TypeScript/React + Canvas API)
- Canvas API でグリッド表示（ズーム・パン対応）
- カラーパレットから色選択
- 塗りつぶし（フラッドフィル）・ペン・消しゴムツール
- アンドゥ / リドゥ

### 3. 糸番号マッピング + 必要量計算
- DMC糸番号データベース内蔵
- グリッド上の色 → 最寄りDMC色番号への自動マッピング
- 各色の使用マス数から糸の必要量（束数）を算出
- 買い物リスト表示

## 技術スタック

| レイヤー | 技術 |
|----------|------|
| 画像処理エンジン | Rust + wasm-pack + wasm-bindgen |
| フロントエンド | TypeScript + React + Vite |
| グリッド描画 | Canvas API |
| 状態管理 | React hooks (useReducer) |
