# 画面設計

## S-1: メイン画面（グリッドエディタ）

```
+----------------------------------------------------------+
|  stitch-studio                          [New] [Import]   |
+----------+-------------------------------+---------------+
|          |                               |               |
| Palette  |     Grid Canvas               | Thread List   |
|          |     (zoom/pan)                |               |
| [色1] ●  |  ┌─┬─┬─┬─┬─┬─┬─┬─┐         | DMC-310  x3   |
| [色2] ●  |  ├─┼─┼─┼─┼─┼─┼─┼─┤         | DMC-321  x2   |
| [色3] ●  |  ├─┼─┼─┼─┼─┼─┼─┼─┤         | DMC-444  x1   |
| [色4] ●  |  ├─┼─┼─┼─┼─┼─┼─┼─┤         | ...           |
|          |  └─┴─┴─┴─┴─┴─┴─┴─┘         |               |
| Tools:   |                               | Total: 6束    |
| [Pen]    |  Zoom: [+][-] 100%           |               |
| [Fill]   |  Grid: 8x4                    |               |
| [Eraser] |                               |               |
+----------+-------------------------------+---------------+
```

**構成要素**:
- 左パネル: カラーパレット + ツール選択
- 中央: Canvas グリッド（ズーム・パン対応）
- 右パネル: 糸番号リスト + 必要量

## S-2: 画像インポートダイアログ

```
+------------------------------------+
|  Import Image                      |
|                                    |
|  [画像プレビュー]                    |
|                                    |
|  Grid Width:  [ 40 ] stitches      |
|  Grid Height: [ 30 ] stitches      |
|  Max Colors:  [ 16 ]               |
|                                    |
|  [Cancel]            [Convert]     |
+------------------------------------+
```

**操作フロー**:
1. ファイル選択 → プレビュー表示
2. パラメータ調整
3. Convert → WASM エンジン呼び出し → グリッドエディタへ
